## Sobre

Somos uma empresa de desenvolvimento de software focada em **entregar soluções que funcionam** — do back-end ao produto final. Trabalhamos com sistemas, aplicações web, consultoria técnica e desenvolvimento de produtos próprios.

> *Código limpo. Arquitetura sólida. Resultado real.*

---

## O que fazemos

- **Sistemas & Back-end** — APIs REST, microsserviços, integrações e automações
- **Web & Apps** — Aplicações modernas com foco em performance e experiência
- **Consultoria Técnica** — Revisão de arquitetura, code review e mentoria
- **Produtos Próprios** — Soluções SaaS e ferramentas para developers
